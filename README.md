<h1>Description</h1>
A single sign-on platform using OAuth.
<h1>Installation</h1>
<h2>Prerequisites</h2>
<ul>
	<li>Web server capable of executing PHP scripts</li>
	<li>PHP version >= 5.2.0</li>
	<li>Support of for the following PHP extensions:
		<ul>
			<li>PDO</li>
			<li>OAuth</li>
		</ul>
	</li>
</ul>
